# Olimps 3D

A project made for participation in https://lata.org.lv/skolas2019

## How to run the server:

```bash
cd server

npm install

npm run start
```

## How to run the site:

1. Navigate to the `client` folder.

2. Specify the server host url in the `/data/config.js` folder.

3. Open `index.html`.
